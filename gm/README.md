# 关于GM命令

在配置文件中的gm的enable配置为true后，会启用配置的http监听，直接GET请求http://127.0.0.1:9092/xxxx?a=1&b=2即可使用。目前支持的GM命令有：

| 终结点      | 参数                      | 备注                                                                  |
|----------|-------------------------|---------------------------------------------------------------------|
| /addcard | player=0&card=1&count=1 | 其中player参数对应**服务器中的**玩家Id，card参数对应协议中的卡牌类型，count参数（非必填，默认1）为增加卡牌的个数 |
